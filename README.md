**Goal:** Go from basic JS knowledge to a competent level in Angular, Node.js, and Software Architecture in 30 working days.
**Intensity:** 9 hours per day (45 hours/week).
**Methodology:** 40/20/40 Blocks (Theory/Break/Project) + Active Recall.

## ðŸŽ¯ Minimum Viable Goals (MVG)

Before moving to the next module, you must meet these requirements:

1. **TypeScript:** Ability to correctly type a small project and understand/resolve compiler errors.
2. **Design Patterns:** Ability to classify patterns and implement at least one from each category (Creational, Structural, Behavioral) in TS.
3. **Node.js (Fundamentals):** Create a functional REST API with Express (GET/POST endpoints).
4. **Angular (Modern):** Build a functional SPA with Standalone Components, Signals, and API consumption.
5. **Node.js (Advanced):** Implement a full CRUD with authentication (JWT) and data persistence.
